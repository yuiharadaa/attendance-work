# Smile Burger 勤怠管理アプリ

## 📝 概要
勤怠打刻・給与計算・勤務履歴可視化を行うWebアプリ。  
HTML/CSS/JavaScriptを用い、Google Apps Script + SpreadsheetをDBとして実装。

## 💡 開発背景
アルバイトなど現場での勤怠管理が煩雑であったことから、  
チーム全員が情報をリアルタイムに共有できる仕組みを構築。
勤怠管理の基本機能は果たした上で、
アルバイトの士気の無さも一緒に改善するという目標のもと「頑張りが可視化される」ことを重視。

## ⚙️ 使用技術
- Frontend: HTML / CSS / JavaScript
- Backend: Google Apps Script (自作API)
- DB: Google Spreadsheet
- Visualization: Chart.js

## 🧩 主な機能
- 出勤・退勤の打刻  
- 勤務履歴・給与の自動計算  
- 勤務データのグラフ表示  
- リアルタイム反映・UI/UX改善
